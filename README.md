# dotfiles

## インストール
```bash
https://github.com/kn-rew/dotfiles.git .dotfiles
sh ~/.dotfiles/install.sh
```

## バックアップ
```bash
sh ~/.dotfiles/backup.sh
git add -A && git commit -m "commit message"
git push
```
